Pour activer l'envoi d'emails :
1. Créez un compte gratuit sur resend.com
2. Ajoutez votre domaine immoconnect-agence.fr
3. Copiez la clé API Resend
4. Dans Supabase → Project Settings → Authentication → SMTP Settings
   Host: smtp.resend.com | Port: 465 | User: resend | Password: [votre clé API]
5. Adresse From: contact@immoconnect-agence.fr (à remplacer par votre vrai email)
