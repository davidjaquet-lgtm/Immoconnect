Tous les produits Checklist
1
Créer et configurer votre compte Stripe
⏱ 20 minutes 💶 Gratuit — Stripe prend 1,5% + 0,25€ par transaction
⚠️
Stripe = votre caisse enregistreuse en ligne
C'est Stripe qui encaisse l'argent de vos clients, le sécurise, et vous le vire sur votre compte bancaire chaque semaine automatiquement. Vous ne touchez jamais directement les CB des clients.
1
Aller sur stripe.com → cliquer "Commencer maintenant"
Créer un compte avec votre email professionnel. Choisir "France" comme pays.
2
Remplir les informations de votre entreprise
Raison sociale ou nom/prénom · Adresse · SIRET si vous en avez un (sinon micro-entrepreneur ou particulier) · Activité : "Services immobiliers / Plateforme en ligne".
3
Vérifier votre identité
Stripe demande une pièce d'identité (CNI ou passeport) + justificatif de domicile. Obligatoire légalement. Prise en charge : 24-48h.
Sans vérification, Stripe bloque les virements au-delà de quelques centaines d'euros.
4
Ajouter votre RIB pour recevoir les virements
Dashboard Stripe → Paramètres → Informations bancaires → Ajouter un compte bancaire → entrer votre IBAN. Les virements arrivent automatiquement chaque semaine.
5
Récupérer vos clés API
Développeurs → Clés API → Copier la Clé secrète (sk_live_...) et la Clé publiable (pk_live_...). Les garder en sécurité — ne jamais les partager.
En attendant la vérification, utilisez les clés TEST (sk_test_...) pour tester sans vrai argent.

3
Créer les liens de paiement Stripe
⏱ 10 minutes ✅ Zéro code

La solution la plus simple : Stripe génère des pages de paiement hébergées, vous n'avez qu'à copier le lien dans vos boutons.

1
Dans Stripe → Liens de paiement → Créer un lien
Sélectionner le produit → Activer "Permettre la promotion des codes" si vous voulez offrir des essais → Cliquer Créer.
2
Copier le lien généré
Stripe génère une URL type https://buy.stripe.com/abc123. Ce lien est une page de paiement complète, sécurisée, RGPD-conforme.

4
Intégrer Stripe dans vos pages
⏱ 30 minutes 🔧 Un peu technique

Si vous voulez une intégration plus poussée (vérifier que le paiement a réussi, débloquer automatiquement l'accès), voici le code :

JavaScript — Redirection vers Stripe Checkout
// Ajouter dans <head> de vos pages
<script src="https://js.stripe.com/v3/"></script>

// Vos Price IDs Stripe (à remplacer par les vrais)
const STRIPE_PRICES = {
  agent_starter:   'price_1ABC...STARTER',
  agent_pro:       'price_1ABC...PRO',
  agent_expert:    'price_1ABC...EXPERT',
  agent_platinium: 'price_1ABC...PLATINIUM',
  acheteur_ess:    'price_1ABC...ACH_ESS',
  acheteur_prem:   'price_1ABC...ACH_PREM',
  invest_pro:      'price_1ABC...INVEST',
  contact_simple:  'price_1ABC...CONTACT',
  exclusivite:     'price_1ABC...EXCLU',
}

const stripe = Stripe('pk_live_VOTRE_CLE_PUBLIABLE')

async function payer(planKey, email = '') {
  const priceId = STRIPE_PRICES[planKey]
  if (!priceId) { alert('Plan introuvable'); return }

  const { error } = await stripe.redirectToCheckout({
    lineItems: [{ price: priceId, quantity: 1 }],
    mode: planKey.includes('contact') || planKey.includes('exclu')
          ? 'payment'      // Paiement unique
          : 'subscription', // Abonnement récurrent
    customerEmail: email,
    successUrl: `https://immoconnect-agence.fr/merci?plan=${planKey}`,
    cancelUrl:  'https://immoconnect-agence.fr/forfaits',
  })

  if (error) console.error(error.message)
}

// Utilisation dans vos boutons :
// <button onclick="payer('agent_pro')">Choisir Agent Pro</button>
// <button onclick="payer('exclusivite')">Acheter l'exclusivité</button>

5
Webhooks — automatiser les actions
⏱ 20 minutes 🔧 Technique

Les webhooks permettent à Stripe d'avertir votre site automatiquement quand un paiement réussit, échoue, ou qu'un abonnement est annulé.

Événement StripeAction automatique à déclencher
checkout.session.completedPaiement réussi → Activer l'accès dans Supabase
customer.subscription.createdNouvel abonné → Envoyer email de bienvenue + leads
customer.subscription.deletedRésiliation → Désactiver l'accès dans Supabase
invoice.payment_failedÉchec de paiement → Email de relance automatique
invoice.paidRenouvellement mensuel → Confirmer l'accès pour 30j
Node.js — backend/webhook.js (à ajouter dans api.js)
const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY)
const endpointSecret = process.env.STRIPE_WEBHOOK_SECRET

// Route webhook — Stripe appelle cette URL à chaque événement
app.post('/api/webhook', express.raw({type: 'application/json'}), async (req, res) => {
  const sig = req.headers['stripe-signature']
  let event

  try {
    event = stripe.webhooks.constructEvent(req.body, sig, endpointSecret)
  } catch (err) {
    return res.status(400).send(`Webhook Error: ${err.message}`)
  }

  switch (event.type) {

    case 'checkout.session.completed': {
      const session = event.data.object
      const email = session.customer_email
      const plan = session.metadata?.plan || 'starter'

      // Activer l'accès dans Supabase
      await supabase.from('abonnements').upsert({
        email, plan, statut: 'actif',
        stripe_customer_id: session.customer,
        debut: new Date().toISOString()
      })

      // Email de bienvenue automatique
      await sendWelcomeEmail(email, plan)
      break
    }

    case 'customer.subscription.deleted': {
      const sub = event.data.object
      // Désactiver l'accès
      await supabase.from('abonnements')
        .update({ statut: 'annule' })
        .eq('stripe_customer_id', sub.customer)
      break
    }

    case 'invoice.payment_failed': {
      const invoice = event.data.object
      // Email de relance
      await sendPaymentFailedEmail(invoice.customer_email)
      break
    }
  }

  res.json({ received: true })
})
